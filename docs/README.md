> 이 게임의 **모든 코드를 초급 웹개발자의 눈높이에서** 풀어 쓴 해설서입니다.
> 예제는 전부 이 저장소의 실제 코드에서 그대로 가져왔어요 — 문서를 읽다가

## 어떻게 읽으면 좋나요?

- 웹 개발이 처음이라면 **01번부터 순서대로** 읽는 걸 추천해요.
- 특정 코드가 궁금해서 왔다면, 아래 표에서 그 주제의 챕터로 바로 가세요.
- 각 챕터는 "개념 설명 → 실제 코드 → 한 줄씩 해설" 순서로 진행됩니다.
- 문서 속 `파일 경로`는 전부 저장소 기준입니다. 열어서 같이 보세요!

## 이 프로젝트의 3가지 대원칙

문서 곳곳에서 계속 만나게 될 원칙이라 먼저 소개할게요.

1. **초당 60번 변하는 것은 캔버스에, 가끔 변하는 것만 React에.**
   점수판(가끔 변함)은 React가, 떨어지는 쓰레기(매 프레임 변함)는 캔버스가 그립니다.
2. **부가 기능이 게임을 죽이면 안 된다.**
   소리·저장·온라인 랭킹이 실패해도 게임은 계속 굴러갑니다. 그래서 코드 곳곳에
   `try { ... } catch { /* 조용히 */ }`가 있어요.
3. **에셋 파일 0개 — 모든 그림과 소리는 코드로.**
   이미지·오디오 파일이 하나도 없습니다. 그림은 `fillRect`로, 소리는 오실레이터로,
   앱 아이콘까지 코드가 만들어요.
