## 3. 플레이 방식 및 멀티플레이어 (Play Mode)

혼자 하는 게임인가요, 아니면 경쟁/협동 게임인가요?

- [ ] 👤 싱글 플레이어 (혼자 플레이하며 최고 점수 기록)
- [ ] 👥 로컬 멀티 플레이어 (하나의 키보드/화면으로 두 명이 플레이)
- [ ] 🌐 실시간 온라인 멀티 플레이어 (웹소켓 기반, 다른 사람과 매칭)
- [x] 🏆 랭킹 시스템만 있는 비동기 경쟁 (게임은 혼자, 점수판은 공유)

## 4. 데이터 저장 방식 (Data Persistence)

사용자의 진행 상황이나 점수를 어떻게 관리할까요?

- [ ] ❌ 저장 안 함 (새로고침하면 완전 초기화되는 가벼운 미니게임)
- [ ] 💾 로컬 스토리지 (현재 브라우저에만 저장, 회원가입 필요 없음)
- [x] ☁️ 데이터베이스 및 로그인 (Supabase)

## 5. 핵심 게임 루프 및 규칙 (Core Gameplay Loop)

생각해두신 게임의 방식이 있다면 자유롭게 적어주세요. (아직 명확하지 않다면 비워두셔도 좋습니다!)

- **목표 (어떻게 점수를 얻나요? / 승리 조건):**
  > 우주 덕후를 타겟팅 한 우주 쓰레기를 먹는 우주 애완 동물을 키우고 진화 시키는 게임

- **패배 조건 (어떻게 하면 게임 오버가 되나요?):**
  > 우주 애완 동물이니까 보살피지 않으면 캐릭터가 죽으면서 게임이 종료됨. 단판으로 게임이 종료 되는 방식은 아님. 특히 지구 괴도를 돌며 우주 쓰레기를 먹고 진화 성장하므로 주인이 있는 국가 상공을 지나게 되면 어드벤테이지를 주는 기획이 필요함.

- **조작 방식 (마우스 클릭, 키보드 방향키, 터치 등):**
  > 화면을 터지하는 곳에 조이스틱이 나타남 우주 공간이니 무조껀 빠르게 움직이 는것이 아니라 분사가스(?)가 유한해서 너무 많이 움직이면 사용자가 조정하지 못하고 그냥 우주 괴도를 돌면서 체력을 회복하거나 그간 수집된 아이템으로 분사가스등을 만들어 플레이 해야함.

## 6. 추가하고 싶은 특별한 기능 (Optional Features)

게임에 들어가면 좋을 것 같은 기능이 있나요?

- [x] 🎵 배경음악 (BGM) 및 타격감 있는 효과음 (SFX)
- [x] 📱 모바일/태블릿 반응형 완벽 지원 (세로/가로 모드 대응)
- [ ] 🌗 라이트 모드 / 다크 모드 전환
- [ ] 💬 게임 내 채팅 기능 (멀티플레이어인 경우)
- [ ] 내 우주펫이 지금 어디에 위치하고 있고 내 머리 위로 언제 지나 가는지 확인 가능
- [ ] 전 세계 우주펫이 얼마나 있고 남은 우주 쓰레기는 얼마나 있으며, 인공위성이 얼마나 있는지 가상 이지만 실 데이터와 어느 정도 썩임으로 인한 진짜 내가 우주 쓰레기를 치우고 있다는 사명감 부여
